The code in this repository was developed for data analysis and processing in a PhD Project, 
Commensal Space Domain Awareness with large parabolic antennas, 2026 by David Smith.

The code was developed in Jupyter Notebook, in Anaconda 3 on python 3.8.8 with the use of the following software packages:

numpy 1.22.4
pandas 1.2.4
matplotlib 3.5.2
skyfield 1.39
astropy 4.2.1
pytz 2021.1
scipy 1.8.1
pyerfa 1.7.3
jplephm 2.15
sys 3.8.8
